# Data-Science-Final-Project
# Overview
Focuses on early detection and prediction of greenhouse gas (GHG) emissions using machine learning techniques
Analyzes environmental, industrial, and atmospheric data to improve emission monitoring
Identifies emission trends and patterns over time
Detects unusual emission spikes for early warning
Supports proactive climate action and policy decision-making
Uses datasets from the National Atmospheric Emissions Inventory (NAEI)

# Tools & Libraries
This project was built using Python and common data science libraries, including:
pandas and NumPy for data handling
Matplotlib and seaborn for visualisation
scikit-learn and XGBoost for modelling

# Models Used
Logistic Regression (baseline model)
Random Forest
XGBoost

# Key Steps
Data cleaning and preprocessing
Feature encoding
Model training and evaluation
Hyperparameter tuning
Threshold optimisation for better detection

# Summary of Results
The results show that ensemble models like Random Forest and XGBoost perform better than the baseline model. They are more effective at detecting emission rises, especially after tuning and threshold adjustments.
